# ADB 工具使用指南

## 项目简介

ADB 工具是一个基于 PySide6 开发的图形化 ADB 命令管理工具，提供了以下功能：

- 设备管理：查看设备状态、连接状态自动检测
- 常用命令：快速执行常用 ADB 命令
- 自定义命令：支持执行多命令、命令历史记录
- 脚本管理：创建、编辑、执行和管理脚本
- 日志管理：查看命令执行结果和历史记录
- 主题管理：支持多种主题切换
- 字体管理：可调整字体大小

## 安装和使用

### 直接运行 EXE 文件

1. 下载并解压 `ADB_Tool.zip` 文件
2. 运行 `ADB_Tool.exe` 即可启动应用
3. 确保您的设备已通过 USB 连接并开启了调试模式

### 从源代码运行

1. 确保已安装 Python 3.7+ 和 PySide6
2. 克隆项目代码
3. 安装依赖：`pip install -r requirements.txt`
4. 运行：`python main.py`

## 功能说明

### 设备管理
- 显示当前连接的设备信息
- 自动检测设备连接状态
- 提供设备操作按钮

### 常用命令
- 提供常用 ADB 命令的快速执行按钮
- 点击按钮即可执行对应命令

### 自定义命令
- 输入框支持多行命令输入
- 支持分号分隔的多命令执行
- 保存命令历史记录
- 可复制命令到剪贴板

### 脚本管理
- **添加脚本**：创建新脚本并保存
- **加载脚本**：从任意位置加载脚本文件
- **编辑脚本**：修改现有脚本
- **删除脚本**：删除不需要的脚本
- **执行脚本**：运行脚本中的所有命令
- **查看详情**：查看脚本的详细信息
- **复制命令**：复制脚本中的所有命令到剪贴板

### 脚本保存建议

建议将脚本保存在 `script` 文件夹中，这样可以确保：
- 脚本在应用启动时自动加载
- 脚本文件不会丢失
- 便于备份和管理

## 构建说明

### 构建 EXE 文件

1. 运行 `build_oneclick.bat` 脚本
2. 构建完成后，可执行文件将生成在 `dist` 目录中

### 构建包含的文件

- `ADB_Tool.exe`：主应用程序
- `README.md`：使用指南
- `script` 文件夹：存储脚本文件

## 注意事项

1. 使用前请确保已安装 ADB 工具并添加到系统 PATH
2. 设备需要开启 USB 调试模式
3. 部分命令需要设备 root 权限
4. 脚本执行可能会影响设备状态，请谨慎使用

## 故障排除

- **设备未检测到**：检查 USB 连接和调试模式
- **命令执行失败**：检查 ADB 安装和设备权限
- **脚本加载失败**：检查脚本文件格式是否正确
- **应用崩溃**：尝试以管理员身份运行
